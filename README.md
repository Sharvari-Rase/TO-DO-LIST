# TO-DO-LIST

A simple to-do list web app built with HTML, CSS, and JavaScript. Add tasks, mark them complete, edit them, or delete them — everything is saved in your browser automatically.

## Features

- Add new tasks
- Edit existing tasks
- Mark tasks as complete (shows a line-through)
- Delete tasks
- Tasks are saved in Local Storage, so they're still there when you refresh the page
- Fully responsive design

## Files

```
todo-list/
├── index.html   -> Page structure
├── style.css    -> Styling
└── script.js    -> App logic
```

## How to Run

1. Download all three files into the same folder.
2. Open `index.html` in any web browser.
3. Start adding tasks!

No installation or setup needed — it's plain HTML, CSS, and JavaScript.

## How to Use

- **Add a task**: Type in the box and click "Add Task" (or press Enter).
- **Complete a task**: Click "Complete" to mark it done. Click "Undo" to mark it not done again.
- **Edit a task**: Click "Edit", change the text, then click "Save".
- **Delete a task**: Click "Delete" to remove it from the list.

## Built With

- HTML5
- CSS3
- Vanilla JavaScript (no frameworks or libraries)
